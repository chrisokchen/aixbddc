# 這個目錄放什麼？

把本 preset 的 handler 實作到某個具體技術堆的機制文件，一個技術堆一檔（如 python-e2e、java-e2e、nextjs-playwright）。

內容只談程式實作設計模式（語言、框架、context 物件、step 檔佈局、matcher 規則）。

# 哪裡會用到此目錄下的檔案？

在 .claude/skills/aibdd-red-execute：variant 被選定後，依對應 `<variant-id>.md`（複製自 `variant.md` 並重新命名，如 `python-e2e.md`）的 runtime 契約把 handler 實作成遵守該技術規範的程式 step definition。
