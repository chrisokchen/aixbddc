# 這個目錄放什麼？

本 preset 的 part-to-DSL plugin。`part_to_dsl.py` 把 parsed spec parts 展開成 DSL instruction template，`__init__.py` 是 package marker。它是「哪個 part 展開成哪些 handler、綁定 target 走哪一種 URI scheme」的 SSOT，每個 boundary preset 都必須有這支 plugin。

# 哪裡會用到此目錄下的檔案？

在 .claude/skills/aibdd-plan 的 `04-dsl-synthesis` Sub-SOP：dsl_cli 的 preset_loader 會動態 exec `part_to_dsl.py` 並呼叫 `generate_templates(parts, context)`，把 spec parts 展開成 DSL skeleton。
