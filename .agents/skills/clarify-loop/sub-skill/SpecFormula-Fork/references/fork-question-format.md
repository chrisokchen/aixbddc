# Fork Question Format

`fork.question` 必須是單一自包含字串，包含四要素：

1. 指出具體 ambiguity / gap
2. 提供 2-4 個具體選項
3. 說明為何這個決策影響下游
4. 明確推薦選項與因果理由

---

## Template

```text
<情境敘述：1-2 句白話文>

<決策問題：1 句>

選項：
- A: <label>
  好處：<...>
  壞處：<...>
- B: <label>
  好處：<...>
  壞處：<...>
[- C ...]
[- D ...]

推薦 <option-id>：<因果解釋>
```

---

## Invariants

- 選項數量必須 2..4（不含 `OTHER`）
- 推薦選項必須存在於清單中
- 文字需可被非技術使用者理解，避免術語堆疊
