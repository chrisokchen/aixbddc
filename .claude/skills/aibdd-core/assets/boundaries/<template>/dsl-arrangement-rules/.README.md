# 這個目錄放什麼？

Gherkin block（Given / Then…）的 DSL 編排推理規章。

# 哪裡會用到此目錄下的檔案？

在 `.claude/skills/aibdd-spec-by-example-analyze` 流程中：`01-example-form-lock` 插入 Example 骨架時，會在 `# @dsl` block 以 `# rule:` 註解嵌入這些檔的路徑；`03-dsl-arrangement` 則依 `# rule:` 路徑讀取規章全文，作為 Given / Then block 編排推理的最高標準，替換 Example 裡的 `<dsl>` placeholder。
