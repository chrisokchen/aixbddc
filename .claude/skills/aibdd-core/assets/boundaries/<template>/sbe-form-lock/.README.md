# 這個目錄放什麼？

放 aibdd-spec-by-example-analyze 階段中做 example-form-lock 自動化所需的宣告檔。`form-lock.profile.yml` 把每個 rule headline 前綴對應到一份 Example 骨架 template，各 `<RuleType類型>.tmpl` 則是待填的 Example / Scenario Outline 骨架。整個目錄是宣告式 boundary preset、不是 runtime truth，只有需要在 SBE 階段做骨架插入的 boundary 才建。

# 哪裡會用到此目錄下的檔案？

在 .claude/skills/aibdd-spec-by-example-analyze 中的 `01-example-form-lock` Sub-SOP 中的腳本，會解析 sbe-form-lock 底下 `form-lock.profile.yml` 然後將底下的 <RuleType類型>.tmpl replace 進 Gherkin 中。
